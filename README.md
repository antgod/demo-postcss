# postcss插件脚手架

仓库提供了gulp与webpack两个版本postcss脚手架。

## gulp
```
$ sudo npm gulp -g
$ cd postcss-plugin-demo && npm i
$ cd ../gulp && npm i
& gulp css
```

在`gulp/dest`生成style.css文件。

## webpack
```
$ sudo npm gulp -g
$ cd postcss-plugin-demo && npm i
$ cd ../webpack && npm i
& webpack
```

在`webpack/dest`生成style.css文件。
